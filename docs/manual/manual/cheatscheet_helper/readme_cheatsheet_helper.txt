Steps to create the cheat sheet:

Modify images

Edit odp file

Export odp file as pdf

Convert pdfs to jpg 

 
pdf2jpg.sh cheatsheet_helper.pdf 

cd cheatsheet_helper.jpg

montage -tile 2x2 -geometry +0+0 001.jpg 003.jpg 002.jpg 004.jpg cheatsheet.jpg


Move the result to manual folder
resize the jpg image to resolution 1500 x 843 px

And convert to pdf as well
